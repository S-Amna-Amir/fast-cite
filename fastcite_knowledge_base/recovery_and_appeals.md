# Recovery Guide: Dealing with Rejections, Disputes & Audits

Something went wrong? Here's how to recover from SECP rejections, FBR audits, or disputes.

---

## PART 1: SECP COMPANY REGISTRATION REJECTED

### Problem: Form 1 Rejected by SECP

**Possible Reasons:**
1. MoA/AoA contains errors
2. Proposed company name unavailable
3. Address proof not acceptable
4. Director/shareholder details incorrect
5. Digital signature invalid or expired

### Recovery Steps

#### Step 1: Check Rejection Notice
- [ ] Read SECP email carefully
- [ ] Identify specific reason (very important)
- [ ] Save all correspondence

#### Step 2: Fix the Issue

**If name is unavailable:**
- [ ] Search for 3 new names on [eservices.secp.gov.pk](https://eservices.secp.gov.pk)
- [ ] Choose one with approval
- [ ] Update MoA & AoA
- [ ] Submit new Form 1

**If MoA/AoA error:**
- [ ] Work with consultant to correct
- [ ] Review side-by-side with SECP template
- [ ] Resubmit corrected version

**If address proof issue:**
- [ ] Verify office address is correct
- [ ] Bring notarized deed if needed
- [ ] Latest utility bill (within 3 months)
- [ ] Resubmit with better documentation

**If digital signature problem:**
- [ ] Visit digital provider again
- [ ] Verify signature code is working
- [ ] May need new signature cert
- [ ] Resubmit Form 1

#### Step 3: Resubmit
- [ ] Go to [eservices.secp.gov.pk](https://eservices.secp.gov.pk)
- [ ] Search for your previous application (reference number)
- [ ] Click "Resubmit" or file new Form 1
- [ ] Pay fees again (non-refundable)
- [ ] Wait 2-3 days for approval

#### Step 4: Timeline Expectations
- First rejection: 1-3 days to receive notice
- Fix time: 2-5 days (depending on issue severity)
- Resubmission processing: 2-3 days
- **Total additional time: 1-2 weeks**

#### Step 5: If Still Stuck
Contact SECP directly:
- **Phone:** [Check SECP website](https://secp.gov.pk)
- **Email:** [Check SECP website](https://secp.gov.pk)
- **In-person:** SECP offices (Islamabad, Karachi, Lahore)
- **For SMC issue?** Ask for SMC specialist help

## PART 2: FBR AUDIT NOTICE

### Problem: You Received Audit Notice from FBR

**What this means:**
FBR selected your business for tax audit. This is normal (random selection).

### Don't Panic - Recovery Steps

#### Step 1: Understand What They Want
- [ ] Read notice carefully
- [ ] List all documents requested
- [ ] Note deadline (usually 30 days)
- [ ] Identify audit scope (1 year? Multiple years?)

#### Step 2: Gather Documents
Prepare:
- [ ] Income and expense records
- [ ] Bank statements (matching audit year)
- [ ] Invoices (issued and received)
- [ ] Receipt book or digital records
- [ ] Proof of tax paid
- [ ] NTN certificate
- [ ] Previous tax returns (last 3 years)
- [ ] Explanation of any large transactions

#### Step 3: Organize by Category
Create folders:
- [ ] Income (invoices proving sales)
- [ ] Expenses (receipts + bills)
- [ ] Deductions (proof of allowed deductions)
- [ ] Bank records (showing deposits/withdrawals)
- [ ] Tax payments (FBR receipts)

#### Step 4: File Response
- [ ] Respond in writing with all documents
- [ ] Attach cover letter explaining
- [ ] Submit on time (before deadline)
- [ ] Keep copy for yourself
- [ ] Get receipt proof from FBR

#### Step 5: Audit Meeting
- [ ] May be called for oral explanation
- [ ] Bring original documents
- [ ] Explain any unusual items
- [ ] Be honest and professional
- [ ] Can bring accountant or tax agent

#### Step 6: Possible Outcomes

**Best case:** Audit passed, no change
**Moderate case:** Minor additions/adjustments + tax payment
**Worst case:** Large income addition + penalties

#### Step 7: If Unfair - Appeal
- **If you disagree,** file appeal:
  - Contact FBR Appeals Cell
  - Submit appeal within 30 days
  - Provide evidence against audit finding
  - May take 6-12 months to resolve

### Prevention Tips for Future
- Keep better records going forward
- Digitize all invoices
- Use accounting app (Wave, Tally)
- Bank transfers instead of cash
- Annual reconciliation with accountant

## PART 3: STRN CHALLENGED / SALES TAX DISPUTE

### Problem: FBR Questions Your Sales Tax Return

**Possible Reasons:**
1. Sales amount inconsistent with prior filings
2. Input tax claimed seems too high
3. Sales tax rate applied incorrectly
4. Missing invoices for claimed sales

### Recovery Steps

#### Step 1: Determine Dispute Type
- [ ] Is it about **sales amount** (too low reported)?
- [ ] Is it about **input tax** (too much claimed)?
- [ ] Is it about **rate applied** (18% vs other)?
- [ ] Is it about **missing evidence**?

#### Step 2: Gather Sales Evidence
- [ ] Invoice copies (all sales)
- [ ] Purchase receipts (for input tax)
- [ ] Bank deposits (proving revenue)
- [ ] Customer contracts/agreements
- [ ] Delivery receipts

#### Step 3: Calculate & Respond
- [ ] Recalculate sales based on documents
- [ ] Explain any changes from prior fili

#### Step 4: File Amended Return
- [ ] Go to [iris.fbr.gov.pk](https://iris.fbr.gov.pk)
- [ ] File amended STRN return
- [ ] Include explanation letter
- [ ] Attach supporting documents
- [ ] Show corrected calculations

#### Step 5: If Demanded Additional Tax
- [ ] Calculate total due
- [ ] Pay within 30 days
- [ ] Submit payment proof with return
- [ ] Request receipt confirmation

#### Step 6: Appeal (If Unfair)
- **If still disagree:** File appeal with FBR
  - Provide evidence
  - Show calculation justification
  - Request review committee hearing

## PART 4: NTN ISSUES

### Problem 1: NTN Registration Rejected

**Causes:**
- CNIC mismatch
- Inactive mobile number
- Website error

**Fix:**
1. Verify CNIC spelling matches exactly
2. Use active mobile number
3. Try again after 24 hours
4. Visit FBR office in person (Islamabad, Karachi, Lahore)

### Problem 2: Filer Status Lost

**Why it happened:**
- Didn't file return by Sept 30
- Return filed but very late

**How to recover:**
1. File return immediately (even if very late)
2. Pay penalties + interest
3. Request reinstatement of filer status
4. FBR restores after verification (2-3 weeks)

## PART 5: PARTNERSHIP DISPUTES

### Problem: Partner Disagrees / Wants to Leave

**If partnership deed is solid:**
1. Refer to partnership deed (exit clause)
2. Follow deed procedure
3. Calculate partner share
4. Register change with Registrar of Firms

**If partnership deed is weak/missing:**
1. Consult partnership lawyer
2. May require court intervention
3. Costs and time will be higher
4. Settlement through negotiation preferred

### Lessons Learned
- Always have written deed
- Clear exit strategy from start
- Regular partner meetings
- Review annually

## PART 6: GENERAL TROUBLESHOOTING

| Problem | Cause | Solution |
|---------|-------|----------|
| Can't log into IRIS/eServices | Password forgotten or account locked | Reset password via email or visit in person |
| Document won't upload | File too large or wrong format | Compress PDF, try different browser |
| Processing time exceeded | SECP/FBR backlog | Contact support, submit inquiry |
| Incorrect tax calculation | Misunderstood slab or deduction | Hire accountant, file corrected return |
| Missing certificate | Lost original | Request duplicate from SECP/FBR (small fee) |
| Audit was unfair | Disagreement on findings | File appeal within deadline |

## PREVENTION CHECKLIST

For future, avoid problems by:
- [ ] Scheduling calendar reminders for Sept 30 (tax deadline)
- [ ] Keeping organized records year-round
- [ ] Filing on time (not late)
- [ ] Double-checking calculations before submitting
- [ ] Using bank transfers (not cash)
- [ ] Semi-annual reconciliation with accountant
- [ ] Partnership deed in writing + notarized
- [ ] Annual partner meetings (if partnership)

---

## WHEN TO HIRE A LAWYER

Consult a tax lawyer if:
- [ ] Audit notice is for large amount
- [ ] Multiple years under review
- [ ] Suspicious activity alleged
- [ ] Partnership dispute escalating
- [ ] Appeal filing required
- [ ] Company deregistration risk
