# FastCite: An AI Legal Assistant for Pakistani Founders (MVP)

## What this does
Simple AI tool that answers business law questions in plain language.

## Features
- SECP registration guidance
- NTN/STN explanation
- Business structure comparison
- Citation-backed answers

## How to run
1. Clone repo
2. Install requirements
3. Run backend
4. Open frontend
